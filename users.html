User Management

Waifus API user management (editing, getting, deleting) docs

1. /users/:id FETCH USER DATA

Method GET
Authorization Basic Auth
Params id: 15 digit user ID
Return User or Error

Gets user data with specified ID (It doesn't return email and password for security!)

2. /users/:id UPDATE USER DATA

Method PUT
Authorization Basic Auth
Params id: 15 digit user ID
Body Data Object
Return Success or Error

Updates specified user's data with given ID

3. /users/:id DELETE USER

Method DELETE
Authorization Basic Auth
Params id: 15 Digit User ID
Body reason: Reason For Deleting
Return Success or Error

Deletes specified user
