Content Management

Waifus API content management (uploading, editing, getting, deleting) docs

1. /(?):id GET CONTENT

Method GET
Authorization None or Basic Auth
Optional Params id?: 15 digit content ID (optional)
Optional Query Params category?: category id
type?: image/video/gif
nsfw?: true/false
limit?: int (ONLY PAID ACCOUNTS)
Return Content or Error

Returns random content with specific filters. Limit filter is only for paid accounts. To use limit filter, must be authorizated.

2. / POST CONTENT

Method POST
Authorization Basic Auth
Body message_url: string
cdn_url: string
verified: bool
verified_by: string
content_owner: string
category: string
upvotes: int
downvotes: int
type: int
nsfw: int
paid_only: bool
post_date: int
verify_date: int
Return Success or Error

Creates a content with specified data

3. /:id EDIT CONTENT

Method PUT
Authorization Basic Auth
Params id: 15 digit content ID
Body Data Object
Return Success or Error

Edits specified content

4. /:id DELETE CONTENT

Method DELETE
Authorization Basic Auth
Params id: 15 Digit Content ID
Body reason: Reason For Deleting
Return Success or Error

Deletes specified content
